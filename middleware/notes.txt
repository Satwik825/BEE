Middleware-
def- middle ware is a function which runs between request and response . It has access to request and response object.(there are 3 arguments- request, response, next)
Running a Middleware
1- at application lever
          app.use([middleware-name])
          to sent request  from one middleware to another, use 'next' argument.
    note- 1-if we run a middleware at application level, then it will run on every request 
          2-middleware run in the order it is called.
